# Tic-Tac-Toe

Tic-Tac-Toe game with:
- Terminal version: `Tic-Tac-Toe.py`
- GUI version (Tkinter): `tic_tac_toe_gui.py`

## Requirements

- Python 3.9+
- Tkinter (usually bundled with Python; on some Linux distributions it may need a separate package)

## Run

```bash
python Tic-Tac-Toe.py
```

```bash
python tic_tac_toe_gui.py
```
