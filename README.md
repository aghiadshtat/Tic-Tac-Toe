# Tic-Tac-Toe
Tic-Tac-Toe game with a desktop GUI built using Tkinter.

## sub header

Run with:

```bash
python Tic-Tac-Toe.py
```

## Features

- Dark, colorful, responsive GUI
- Start page with:
  - Start vs Player
  - Start vs Bot (Hard / Medium / Easy)
  - Exit
- Game logic separated from GUI code
- Turn indicator dot (green/red) for active player
- Move sound on every move
